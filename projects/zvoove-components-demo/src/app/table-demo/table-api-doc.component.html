*zvTableColumnTemplate="let item" Can be used to define a custom column template. Needs to be inside a zv-table-column. *zvTableColumnHeaderTemplate Can be used to define a custom column header template. Needs to be inside a zv-table-column. *zvTableRowDetailTemplate="let item" Neccessary to define a row detail template. Needs to be inside a zv-table-row-detail. *zvTableTopButtonSection Can be used to add buttons to the header of the table. Needs to be inside a zv-table. *zvTableCustomHeader Can be used to add custom stuff to the table header. Needs to be inside a zv-table. *zvTableCustomSettings Can be used to add custom stuff to the table settings. Needs to be inside a zv-table.
